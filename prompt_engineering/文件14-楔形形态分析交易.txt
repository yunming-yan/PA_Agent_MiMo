【文件定位】本文件职责：识别楔形/三推形态，判断动能衰减与反转信号。仅在诊断检测到楔形/三推时加载。

【核心知识】

一、楔形的本质定义

1. 楔形（Wedge）的根本定义
   - 楔形 = 三推（Three Pushes）形态
   - 三推是指价格三次向同一方向推进，每次推进的幅度递减
   - 趋势线（Trendline）和通道线（Channel Line）逐渐收敛
   - 本质是方向性动能在三次推进中逐渐衰减

2. 为什么是三推？
   - 第一次推：初始动量最强，趋势的尖峰（Spike）阶段
   - 第二次推：动量开始减弱，但仍有足够力量创新高/低
   - 第三次推：动量显著不足，收敛明显，是最后的挣扎
   - 三推之后，动能耗尽，市场大概率反转或深度回撤

3. 楔形的收敛特征
   - 每一推的高度/深度小于前一推
   - 趋势线（连接各推的起点）和通道线（连接各推的终点）逐渐靠拢
   - 如果两条线不收敛，则不是楔形，可能只是通道

二、上升楔形与下降楔形

1. 上升楔形（Rising Wedge）
   - 形态：价格逐步走高，但每一推的幅度递减
   - 趋势线和通道线都向上倾斜，但通道线斜率小于趋势线
   - 交易含义：看跌反转形态（Bearish Reversal Pattern）
   - 突破方向：向下突破概率>60%
   - 为什么看跌？因为上升动能在每次推进中减弱，买方力量衰竭

2. 下降楔形（Falling Wedge）
   - 形态：价格逐步走低，但每一推的幅度递减
   - 趋势线和通道线都向下倾斜，但通道线斜率小于趋势线
   - 交易含义：看涨反转形态（Bullish Reversal Pattern）
   - 突破方向：向上突破概率>60%
   - 为什么看涨？因为下降动能在每次推进中减弱，卖方力量衰竭

3. 重要区别：楔形方向 ≠ 突破方向
   - 上升楔形 → 向下突破（反转看跌）
   - 下降楔形 → 向上突破（反转看涨）
   - 这是初学者最容易混淆的概念

4. 楔形与三角形的区别
   - 三角形（Triangle）：一条线水平，一条线倾斜，通常是对称三角形或直角三角形
   - 楔形：两条线都倾斜，且方向相同
   - 上升三角形：上边水平，下边上升 → 看涨延续
   - 下降三角形：下边水平，上边下降 → 看跌延续
   - 上升楔形：两边都上升 → 看跌反转
   - 下降楔形：两边都下降 → 看涨反转
   - 实际交易中，三角形和楔形的交易方法非常相似，关键都在于收敛后的突破方向

三、楔形回撤 vs 楔形反转

这是Al Brooks体系中关于楔形最重要的区分：

1. 楔形回撤（Wedge Pullback）
   - 定义：在顺势方向上出现的小型楔形，通常只持续10-20根K线
   - 方向：与主趋势方向相反的楔形
   - 例如：上涨趋势中出现下降楔形 → 下降楔形是上涨趋势中的回撤
   - 交易含义：顺势入场机会！楔形回撤结束后，价格大概率继续原趋势方向
   - 入场：楔形回撤突破时顺势入场
   - 目标：前高/前低或更高

2. 楔形反转（Wedge Reversal）
   - 定义：在趋势末端出现的大型楔形，至少20根K线，且是逆势架构
   - 方向：与主趋势方向相同的楔形
   - 例如：上涨趋势末端出现上升楔形 → 上升楔形是趋势反转信号
   - 交易含义：趋势可能反转！但需要确认
   - 入场：楔形突破后等待回撤再入场（二次入场）
   - 目标：楔形起点 + 测量运动

3. 关键区别对照

| 特征 | 楔形回撤 | 楔形反转 |
|------|---------|---------|
| K线数量 | 10-20棒 | ≥20棒 |
| 架构性质 | 顺势架构 | 逆势架构 |
| 方向 | 与趋势相反的楔形 | 与趋势同向的楔形 |
| 突破后 | 回到原趋势方向 | 趋势反转 |
| 入场方式 | 突破时直接入场 | 最好等回撤入场 |
| 概率 | 顺势延续概率>65% | 反转概率约55-60% |
| 目标 | 前极点或更高 | 楔形起点+测量运动 |

4. 判断标准
   - 首先确定主趋势方向（看HTF）
   - 楔形方向与主趋势相反 → 楔形回撤
   - 楔形方向与主趋势相同 → 楔形反转
   - 不确定时，以K线数量辅助判断：<20棒偏向回撤，>20棒偏向反转

四、抛物线楔形（Parabolic Wedge）

1. 定义
   - 价格在趋势末期加速上涨/下跌，形成抛物线形状
   - 本质上是一个加速的楔形——三推中的每一推越来越快（而非越来越慢）
   - 这是楔形的变异形式，与标准楔形的收敛特征相反

2. 抛物线楔形 = 买入高潮（Buy Climax）/ 卖出高潮（Sell Climax）
   - 上升抛物线楔形 = 买入高潮：最后的疯狂买入，动能在极端水平
   - 下降抛物线楔形 = 卖出高潮：最后的恐慌卖出，动能在极端水平

3. 交易含义
   - 抛物线楔形是趋势即将结束的强烈信号
   - 但不是立刻反转——通常先进入交易区间，之后可能反转也可能继续
   - 突破抛物线楔形后，通常有深回撤
   - 深回撤后可能二次测试极点

4. 交易策略
   - 不要在抛物线楔形中追趋势——入场太晚，风险回报比极差
   - 如果已持有顺势仓位，在抛物线加速时开始减仓
   - 等待抛物线楔形突破后，考虑逆势交易
   - 逆势交易需要二次入场确认

五、微楔形（Micro Wedge）

1. 定义
   - 由3-4根K线组成的极小楔形
   - 三推中的每一推可能只有1根K线
   - 本质是短期的动能衰减信号

2. 识别特征
   - 3-4根K线中，每根K线的高点/低点逐渐靠近
   - 可能在趋势线附近出现
   - 通常是更大形态的一部分

3. 交易含义
   - 微楔形是短期的回撤/反转信号
   - 在趋势中，微楔形可能只是暂时休整
   - 在交易区间中，微楔形可能是边界反转的一部分

4. 交易策略
   - 微楔形通常不单独作为入场依据
   - 与其他信号结合使用（如支撑/阻力位、EMA等）
   - 作为仓位管理的辅助信号（如减仓、移动止损）

六、楔形交易规则详解

1. 入场规则
   a) 楔形回撤入场
   - 条件：主趋势明确 + 出现与趋势相反的楔形 + K线数<20
   - 时机：楔形突破趋势线时入场
   - 优先级：高（顺势交易，概率优势明显）

   b) 楔形反转入场
   - 条件：趋势末端 + 出现与趋势同向的楔形 + K线数≥20
   - 时机：最好等待楔形突破后回撤再入场（二次入场）
   - 优先级：中（逆势交易，需要更多确认）
   - 特例：如果反转架构异常强（大实体反转棒、缺口等），可在突破时直接入场

   c) 抛物线楔形反转入场
   - 条件：抛物线加速后出现突破
   - 时机：突破后等深回撤，在二次入场点入场
   - 优先级：中低（需要耐心等待，但回报可能很大）

2. 目标规则
   a) 楔形回撤目标
   - 第一目标：楔形起点（回撤前的极点）
   - 第二目标：楔形起点的等幅测量运动
   - 第三目标：根据更大结构确定（如通道对边、前高/低等）

   b) 楔形反转目标
   - 第一目标：楔形起点
   - 第二目标：楔形高度的测量运动（从突破点向下/上量楔形高度）
   - 注意：楔形反转后的测量运动通常是楔形高度的1-2倍

   c) 计算方法
   - 测量运动（Measured Move）= 楔形最高点到最低点的距离
   - 从突破点开始，沿突破方向等距离投射

3. 失败处理
   a) 楔形回撤失败
   - 表现：突破后价格又回到楔形内，并继续回撤
   - 处理：止损出场，重新评估趋势是否仍在
   - 可能原因：楔形不是回撤而是反转的早期阶段

   b) 楔形反转失败
   - 表现：突破后价格很快回到楔形内，趋势继续
   - 处理：止损出场，承认反转失败
   - 可能原因：趋势仍然强劲，楔形只是趋势中的停顿

   c) 抛物线楔形失败
   - 表现：突破后价格重新加速创新高/低
   - 处理：止损出场，等待下一个高潮信号
   - 注意：抛物线楔形失败后可能形成更大的抛物线

【分析流程】

步骤1：识别潜在楔形
- 观察价格是否出现三推结构
- 数推数：第一推、第二推、第三推
- 画趋势线和通道线，判断是否收敛

步骤2：确定楔形类型
- 上升楔形还是下降楔形？
- 微楔形（3-4棒）还是标准楔形（>10棒）？
- 是否是抛物线楔形（加速而非收敛）？

步骤3：判断楔形性质
- 确定主趋势方向（看HTF）
- 楔形方向与趋势相反 → 楔形回撤
- 楔形方向与趋势相同 → 楔形反转
- K线数量辅助：<20棒偏向回撤，≥20棒偏向反转

步骤4：评估楔形质量
- 收敛是否明显？（越明显，信号越可靠）
- 三推是否清晰？（每推递减，推数明确）
- 是否在关键位置？（支撑/阻力、EMA、前高/低等）
- 成交量是否配合？（每推成交量递减更好）

步骤5：制定交易计划
- 根据楔形性质（回撤/反转）选择入场策略
- 确定入场价位和止损位置
- 计算目标位（楔形起点 + 测量运动）
- 评估风险回报比

步骤6：执行与跟踪
- 等待楔形突破
- 突破后决定直接入场还是等回撤
- 入场后设置止损和目标
- 跟踪走势，根据实际情况调整

【实战规则】

一、楔形回撤交易规则（高概率交易）

1. 上涨趋势中的下降楔形回撤
   - 入场：楔形向上突破时买入
   - 止损：楔形最低点外1跳动
   - 目标1：楔形起点（前期高点）
   - 目标2：楔形起点 + 楔形高度的测量运动
   - 加分项：楔形回撤至EMA附近获得支撑

2. 下跌趋势中的上升楔形回撤
   - 入场：楔形向下突破时卖出
   - 止损：楔形最高点外1跳动
   - 目标1：楔形起点（前期低点）
   - 目标2：楔形起点 + 楔形高度的测量运动

二、楔形反转交易规则（中等概率交易）

1. 上涨趋势末端的上升楔形反转
   - 入场：楔形向下突破后，等待回撤至楔形下边再卖出
   - 止损：回撤高点外1跳动（或楔形最高点外1跳动）
   - 目标1：楔形起点
   - 目标2：楔形高度测量运动
   - 注意：如果反转信号异常强（大阴线、缺口等），可直接入场

2. 下跌趋势末端的下降楔形反转
   - 入场：楔形向上突破后，等待回撤至楔形上边再买入
   - 止损：回撤低点外1跳动（或楔形最低点外1跳动）
   - 目标1：楔形起点
   - 目标2：楔形高度测量运动

三、抛物线楔形交易规则

1. 识别抛物线楔形
   - 价格加速创新高/低
   - 每一推的斜率增大（而非收敛）
   - 通常伴随极大成交量和极端动量指标

2. 交易策略
   - 不追趋势！
   - 已有仓位开始减仓
   - 等待突破后的深回撤
   - 在二次入场点考虑逆势交易
   - 止损设在抛物线极点外1跳动

四、微楔形交易规则

1. 作为辅助信号
   - 在趋势中出现微楔形 → 趋势可能暂时停顿
   - 在支撑/阻力位出现微楔形 → 增加反转概率
   - 不单独作为入场依据

2. 结合使用
   - 微楔形 + EMA支撑/阻力 → 入场信号
   - 微楔形 + 双重顶/底 → 增强反转信号
   - 微楔形 + 突破失败 → 顺势入场信号

【例外与陷阱】

1. 楔形识别陷阱
   - 假三推：不是所有三次推进都是楔形，必须有收敛特征
   - 过早数推：第二推还没完成就假设会有第三推
   - 强行画线：趋势线和通道线不自然地凑成楔形

2. 楔形方向陷阱
   - 上升楔形不一定向下突破：约35-40%的上升楔形向上突破（趋势延续）
   - 下降楔形不一定向上突破：约35-40%的下降楔形向下突破（趋势延续）
   - 在强趋势中，楔形可能只是趋势的停顿而非反转

3. 楔形回撤 vs 反转陷阱
   - 将楔形反转误判为楔形回撤：在趋势末端的小楔形可能不是回撤
   - 将楔形回撤误判为楔形反转：在趋势中的大楔形可能不是反转
   - 解决方法：结合HTF趋势方向和K线数量综合判断

4. 抛物线楔形陷阱
   - 过早判断高潮：看似抛物线楔形但趋势继续加速
   - 反转幅度不足：抛物线楔形突破后可能只回撤到起点，不形成真正的反转趋势
   - 二次测试陷阱：深回撤后可能二次测试极点并创新高/低

5. 微楔形陷阱
   - 过度解读：3-4根K线的微楔形信息量有限
   - 忽略大结构：微楔形可能只是大楔形中的一推
   - 频繁交易：看到微楔形就交易，导致过度交易

6. 楔形与三角形混淆
   - 上升三角形是看涨延续形态（不是看跌反转）
   - 上升楔形是看跌反转形态（不是看涨延续）
   - 混淆两者会导致方向性错误

7. 通用陷阱
   - 楔形目标不达：楔形突破后可能无法到达测量运动目标
   - 假突破楔形：楔形边界被突破后迅速回到楔形内
   - 时间陷阱：楔形可能在收敛过程中持续很久，提前入场容易被反复止损
