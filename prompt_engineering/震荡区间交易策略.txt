【文件定位】本文件职责：在已确认震荡区间中，给出低买高卖刮头皮的具体方案。在诊断结果为震荡区间（trading_range）或趋势型交易区间（trending_tr）时加载。

⚠️ 【trending_tr vs trading_range 分支处理】
trending_tr（趋势型交易区间）与普通 trading_range（无方向交易区间）使用本文件同一策略框架，但有以下关键差异必须注意：

| 特征 | trading_range | trending_tr |
|---|---|---|
| 方向 | 无明显方向偏好 | 有偏多或偏空方向 |
| 边界测试 | 对称测试上下边界 | 不对称测试（顺势方向边界测试更多） |
| 均线 | 平坦，价格反复穿越 | 倾斜，价格在均线一侧停留更久 |
| 交易策略 | 双向刮头皮，上下边界平等 | 顺势方向为主（70%仓位），逆势方向为辅（30%仓位） |
| 突破概率 | 上下突破概率相等 | 顺势方向突破概率更高（约60%） |
| 逆势交易 | 上下边界均可做 | 逆势方向只在极点反转信号强时做，且仓位减半 |

当 cycle_position="trending_tr" 时，执行以下调整：
1. 优先做顺势方向的边界反转（如 direction=bullish，优先在区间下边界做多）
2. 逆势方向仅在出现强反转信号（如楔形+双棒反转）时才考虑，且仓位不超过顺势方向的50%
3. 区间突破信号只跟顺势方向，逆势方向突破默认视为假突破
4. 顺势方向止盈可稍远（区间宽度的60-80%），逆势方向止盈收紧至区间宽度的30-50%

【核心知识】

一、核心格言详解："低买、高卖、刮头皮"

这是Al Brooks体系在交易区间中的核心策略，每一个词都有精确的含义：

【低买（Buy Low）】
- 不是"在最低点买"，而是在区间下边界附近买
- 下边界是支撑位，价格在此反弹的概率高于继续下跌
- 具体做法：等待价格到达下边界 + 出现看涨反转信号（如反转棒、外包棒）时买入
- 止损设置：下边界下方1~3跳动（超过下边界意味着支撑失败）
- 止盈目标：区间上边界（或更保守：中间位置）

【高卖（Sell High）】
- 不是"在最高点卖"，而是在区间上边界附近卖
- 上边界是阻力位，价格在此回落概率高于继续上涨
- 具体做法：等待价格到达上边界 + 出现看跌反转信号（如反转棒、外包棒）时卖出
- 止损设置：上边界上方1~3跳动
- 止盈目标：区间下边界（或更保守：中间位置）

【刮头皮（Scalp）】
- 利润目标小，通常是区间宽度的20%~50%
- 不追求"吃掉整个区间"，而是积累多次小幅盈利
- 每次刮头皮的目标是：赚取的跳动数 > 止损跳动数
- 刮头皮的核心是胜率——多次小幅盈利累积成显著收益
- 典型刮头皮：止损5跳动，止盈8跳动；胜率55%即可盈利

刮头皮 vs 波段交易（关键区分）：
- 刮头皮：区间中的短期反向运动，目标小，胜率要求高
- 波段交易：趋势中的大波段，目标大，胜率要求相对低
- 在区间中做波段交易（持有等待大利润）是亏损的主要原因
- 在趋势中做刮头皮（过早止盈）会错失大利润

二、止损单入场策略

止损单（Stop Order）是初学者在区间交易中的首选入场方式。止损单的特点是：等待市场先给出方向信号，然后以止损单的形式入场，止损位置预先设定。

【策略1：区间底部High 2买入】
适用场景：价格在区间下边界附近
操作步骤：
1. 确认当前处于交易区间状态
2. 等待价格到达区间下边界
3. 等待出现"高2"形态（价格创新低后，第二轮上攻未能突破前一轮高点）
4. 在第二高点下方1跳动处放置止损买单
5. 止损位置：区间下边界下方1跳动
6. 止盈目标：区间上边界或更近的中间位置

原理：高2是多头开始反攻的信号，第二高点是入场的好位置。

【策略2：区间顶部Low 2卖出】
适用场景：价格在区间上边界附近
操作步骤：
1. 确认当前处于交易区间状态
2. 等待价格到达区间上边界
3. 等待出现"低2"形态（价格创新高后，第二轮下探未能突破前一轮低点）
4. 在第二低点下方1跳动处放置止损卖单
5. 止损位置：区间上边界上方1跳动
6. 止盈目标：区间下边界或更近的中间位置

原理：低2是空头开始反攻的信号，第二低点是入场的好位置。

【策略3：反转棒确认后止损单入场】
操作步骤：
1. 价格到达区间边界
2. 出现反转棒（RB）：与当前趋势方向相反的强K线
3. 等待下一根K线收盘确认（确认棒）
4. 在反转棒极点外1跳动处放置止损单
5. 止损：反转棒极点外1跳动（总止损2跳动）

适用性：这是最保守的入场方式，适合新手和高度不确定的市场。

【策略4：外包棒（OB）止损单入场】
操作步骤：
1. 价格到达区间边界
2. 出现外包棒（OB）：当前K线完全包含前一根K线
3. 等待市场选择方向（下一根K线突破OB的高点或低点）
4. 在OB的高点上方（做多）或OB的低点下方（做空）放置止损单
5. 止损：OB的另一侧1跳动

三、限价单入场策略

限价单（Limit Order）是在区间边界预先挂单，等待价格"回来找你"。与止损单不同，限价单不需要等待确认——你在价格到达边界时就提前挂好单。

【策略1：空头尖峰时买入（卖空尖峰处买入）】
适用场景：下跌趋势尾端，出现空头尖峰（快速下跌后，价格开始企稳）
操作：
1. 在尖峰底部（最低点）上方1~2跳动处挂限价买单
2. 止损：尖峰最低点下方1跳动
3. 逻辑：空头尖峰是卖压的集中释放，之后通常会有反弹

【策略2：多头尖峰时卖出（买多尖峰处卖出）】
适用场景：上涨趋势尾端，出现多头尖峰
操作：
1. 在尖峰顶部（最高点）下方1~2跳动处挂限价卖单
2. 止损：尖峰最高点上方1跳动
3. 逻辑：多头尖峰是买盘的集中释放，之后通常会有回调

【策略3：测量移动目标位限价单】
操作：
1. 计算区间前的运动幅度（从起涨点到区间的距离）
2. 如果区间突破向上，目标位 = 区间上边界 + 该运动幅度
3. 在目标位附近（稍下方）挂限价卖单
4. 止损：目标位上方1~2跳动

适用性：测量移动目标是"磁铁"，价格倾向于朝这些位置运动后回落。

【限价单 vs 止损单的对比】
限价单优点：可以提前挂单，不需要盯盘；可能获得更好的入场价格
限价单缺点：如果市场直接越过边界，可能错失机会；也可能接到"飞刀"
止损单优点：确认信号后入场，更可靠
止损单缺点：需要盯盘；可能在确认时价格已经移动了一段

建议：初学者使用止损单，有经验后逐步引入限价单。

四、带刺铁丝（Barbwire）的交易规则

Barbwire是交易区间中的最高密度形态，需要特殊的交易规则：

规则1：不在Barbwire中交易
在Barbwire形成过程中，方向完全不可预测。正确的做法是等待Barbwire形成结束，等待价格以尖峰形式突破Barbwire。

规则2：Barbwire突破方向由HTF决定
Barbwire之后的价格运动方向，通常由大时间框架的趋势方向决定。在HTF上升趋势中，Barbwire向上突破的概率更高；在HTF下降趋势中，向下突破的概率更高。

规则3：等尖峰突破Barbwire
Barbwire的突破必须是尖峰级别的——至少3~5根同方向连续趋势棒。普通幅度的突破（只有1~2根趋势棒）大概率会被拉回Barbwire。

规则4：Barbwire突破后等回踩测试
Barbwire突破后，几乎必然会有回踩测试（Breakout Test）。等待回踩测试成功后再入场，胜率远高于在突破瞬间追入。

Barbwire交易的具体流程：
1. 识别Barbwire（紧凑横向整理）
2. 等待尖峰突破Barbwire（方向由HTF决定）
3. 等待回踩测试（价格回到突破点附近）
4. 回踩测试成功（价格在突破点获得支撑/阻力）
5. 顺势入场

五、三角形交易规则

【上升三角形交易】
入场规则：
1. 如果向上突破（上边界被尖峰突破）：等待回踩测试，回踩不破上边界则买入
2. 如果向下跌破（低点渐高的趋势线被跌破）：等待反弹，反弹到均线或趋势线附近做空

止盈设置：
- 向上突破后的目标：测量运动目标（区间高度向上延伸）
- 向下突破后的目标：测量运动目标（区间高度向下延伸）

【下降三角形交易】
入场规则：
1. 如果向下突破（下边界被尖峰跌破）：等待回踩测试，回踩不上破下边界则做空
2. 如果向上突破（高点渐低的趋势线被突破）：等待回踩，回踩不上破趋势线则买入

止盈设置：
- 向下突破后的目标：测量运动目标（区间高度向下延伸）
- 向上突破后的目标：测量运动目标（区间高度向上延伸）

【对称三角形交易】
等待边界收窄到极致
等待尖峰突破
等待回踩测试
顺势入场

【扩张三角形交易】
不交易

六、时间因素：中间时段 + 中间位置 = 不交易

时间因素是区间交易中常被忽视但非常重要的因素。

【时间框架的时间效应】
- 在日内交易中，某些时段天然具有更高的波动性和方向性（如开盘30分钟）
- 某些时段天然低波动（如中午12点~14点的美国时段）
- 在低波动时段交易区间，边界反弹的概率更低

【"中间时段 + 中间位置"规则】
"中间时段 + 中间位置 = 不交易"是最重要的禁区规则。

含义：
- 中间时段：低流动性、低波动性的时段
- 中间位置：区间中间区域，不是边界附近
- 在这两个条件同时满足时入场，刮头皮的成功率显著降低

具体场景：
- 下午2点~4点（美国午后低波动期）+ 区间中间位置 → 不交易
- 任何时段的区间中间1/3区域 → 不交易

正确做法：只在边界附近 + 高波动时段交易

七、刮头皮 vs 波段的选择

这是区间交易中最重要的时间框架选择问题。

【刮头皮（ scalp）】
定义：持仓时间从几分钟到几十分钟，目标利润是区间宽度的20%~50%
适用场景：区间边界附近的短期反弹/回落
胜率要求：高（通常需要55%以上）
风险回报比：通常1:1到1:2
操作频率：高（多次交易）

【波段交易（Swing / Position）】
定义：持仓时间为数小时到数天，目标利润是区间宽度的80%~100%
适用场景：区间突破后的大波段（但要注意：80%突破失败）
胜率要求：相对低（40%~50%）
风险回报比：通常1:3或更高
操作频率：低（等待机会）

【选择原则】
1. 在区间中，默认选择刮头皮（不追求大利润）
2. 只有当出现尖峰突破（>80%概率成功）时，才考虑波段
3. 如果你选择了波段，刮头皮单必须取消（避免两个都成交）
4. 如果你选择了刮头皮，不要因为"还想赚更多"而改成波段（贪婪是亏损的根源）

八、止损和止盈设置

【止损设置标准】
1. 标准止损：信号棒极点外1跳动（这是初学者的唯一选择）
2. 最小止损：不能小于1跳动（否则胜率会大幅下降）
3. 最大止损：区间宽度的10%（超过则这笔交易的风险回报比太差）
4. 突破止损：如果是止损单入场，止损在边界外1跳动

止损类型的选择：
- 止损单入场：止损在边界外1跳动（固定风险）
- 限价单入场：止损在入场价下方一定距离（根据市场波动性调整）

【止盈设置方法】
1. 固定比例法：区间宽度的50%~80%
2. 测量移动法：区间前波段的等距目标
3. 边界法：到区间另一边边界
4. 分批了结法：50%仓位在中间位置，50%到另一边边界

【交易者方程在止损止盈中的应用】
交易者方程：胜率 × 回报 > 败率 × 风险

举例（区间下边界买入）：
- 止损距离：5跳动（下边界下方1跳动）
- 止盈距离：10跳动（区间中间位置）
- 胜率：55%（基于区间边界的历史反弹概率）
- 败率：45%
- 计算：0.55 × 10 = 5.5 > 0.45 × 5 = 2.25 → 通过

如果止盈距离只有5跳动（区间宽度的30%）：
- 0.55 × 5 = 2.75 > 0.45 × 5 = 2.25 → 勉强通过（但摩擦成本会让结果变负）

结论：刮头皮止盈至少要到区间宽度的50%以上，才值得交易。

【关键规则：不要移动止损】
- 入场后，不要因为"害怕被扫"而提前移动止损
- 不要因为"感觉价格会回来"而扩大止损
- 止损只可以在盈利方向上移动（收紧），绝不能在亏损方向上移动（放宽）
- 放宽止损等于承认错误，应该直接平仓


【分析流程】

当你准备在交易区间中进行交易时，请按以下步骤执行：

步骤1：确认区间状态
- 识别区间边界（上边界和下边界）
- 评估区间密度（高密度=刮头皮好机会）
- 检查是否Barbwire（如果是，等待）

步骤2：确定交易方向
- 当前价格在区间中的位置（靠近上边界→考虑做空，靠近下边界→考虑做多）
- 是否有反转信号？（反转棒？外包棒？）
- 大时间框架的方向偏好（如果HTF是上涨，靠近下边界做多更安全）

步骤3：选择入场方式
- 止损单：等待信号确认后入场（保守，适合新手）
- 限价单：提前挂单在边界附近（激进，适合有经验者）

步骤4：计算交易者方程
- 止损距离：___跳动
- 止盈距离：___跳动
- 预期胜率：___%
- 胜率×回报 vs 败率×风险 → 通过/不通过

步骤5：执行交易
- 设定止损
- 设定止盈
- 评估持仓时间（刮头皮 vs 波段）
- 记录交易计划

步骤6：持仓管理
- 监控价格是否朝有利方向移动
- 如果盈利，可以考虑将止损移到盈亏平衡点
- 如果市场状态改变（变成趋势），评估是否调整策略


【实战规则】

规则1：只在边界附近交易
区间中间位置没有任何交易价值。在边界附近等待——不交易是一种决策，耐心等待边界机会是专业交易者的标志。

规则2：止损是唯一的保护
区间中的刮头皮是高频率交易，高频率意味着高错误率。没有止损保护，一笔错误就会吞噬多次盈利。每一笔交易都必须有止损。

规则3：不在Barbwire中交易
Barbwire是方向最不明确的状态。在Barbwire中交易等于在黑暗中被蒙眼飞行。等待Barbwire结束，等待尖峰突破，等待回踩测试。

规则4：突破后等测试
"追突破"是区间交易中最常见的亏损模式。正确的做法：突破后不追，等待回踩测试。如果测试成功，顺势入场；如果测试失败，放弃或反向交易。

规则5：刮头皮止盈目标不低于区间宽度的50%
如果你的止盈目标低于区间宽度的50%，交易者方程大概率不通过（除非你的胜率>70%）。设定合理的止盈目标，是盈利的前提。

规则6：分批了结
不要试图在最高点/最低点全仓了结。50%仓位在中间位置了结，50%仓位持有到另一边边界。这样既能锁定部分利润，又能给大利润留出空间。

规则7：中午时段减少交易
美国东部时间中午12点~下午2点是低波动期，区间边界反弹的概率降低。如果在此时段交易，降低仓位或放弃机会。

规则8：识别区间中的趋势并跟随
如果区间中出现了明确的趋势（虽然整体仍是区间），可以短暂地"跟随趋势"，但目标只是到区间另一边边界，不要把趋势当成真的趋势来追逐。


【例外与陷阱】

陷阱1：在区间中间追单
"我看到价格朝边界移动了，赶紧追进去"——这是最常见的区间交易错误。追入的风险是：价格可能已经到了边界的另一侧，你的止损空间极小，回报空间也极小。

陷阱2：把刮头皮做成波段
"已经赚了这么多，再等等，看看能不能赚更多"——这是贪婪的陷阱。区间中刮头皮赚的是"确定的20%~50%"，变成波段后面临"80%突破失败"的风险。守住纪律，到了目标就了结。

陷阱3：不设止损
"我会看着盘的，不需要止损"——这是对人类弱点的忽视。市场会制造让你不愿意平仓的情况（锁仓恐惧、损失厌恶），止损是唯一的保护。

陷阱4：在低密度区间中频繁交易
低密度区间的边界不可靠，很多"边界"只是价格的偶然波动，不是真正的支撑/阻力。在低密度区间中降低交易频率，等待更高质量的信号。

陷阱5：在扩张三角形中交易
扩张三角形的边界向外扩散，方向极度不明。在这种形态中，边界只是暂时的，价格随时可能朝任何方向爆发。不交易是最佳选择。

例外1：连续两天的低波动区间
如果同一个品种连续两天处于低波动区间，第三天开盘出现尖峰突破的概率增加。这是已知的高概率事件，可以提前准备。

例外2：区间边界与均线重叠
当均线恰好穿过区间边界时，这个位置的支撑/阻力意义大幅增加。这是刮头皮的高概率机会，可以适当提前入场。
